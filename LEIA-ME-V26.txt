OFERTA CERTA V26 — FRETE AUTOMÁTICO COM MELHOR ENVIO

O QUE MUDOU
- Remove o frete fixo.
- Cadastra peso, comprimento, largura e altura em cada produto próprio.
- Cliente digita o CEP no carrinho.
- Sistema consulta opções reais no Melhor Envio.
- Cliente escolhe transportadora, preço e prazo.
- O servidor recalcula o frete antes de abrir o Mercado Pago.
- O frete aparece separado no Checkout Pro.

1. SUPABASE
Execute setup_v26.sql no SQL Editor.

2. GITHUB
Substitua os arquivos do projeto pelos arquivos deste ZIP.
Mantenha outros arquivos antigos da pasta api que não tenham o mesmo nome.

3. VERCEL — NOVAS VARIÁVEIS
Adicione:
MELHOR_ENVIO_TOKEN = token do Melhor Envio
MELHOR_ENVIO_API_URL = https://sandbox.melhorenvio.com.br
MELHOR_ENVIO_USER_AGENT = Oferta Certa (seu-email@exemplo.com)
SHIPPING_ORIGIN_ZIP = seu CEP de postagem, somente números

Para produção, depois de concluir testes e autorização:
MELHOR_ENVIO_API_URL = https://www.melhorenvio.com.br

As variáveis anteriores do Mercado Pago e Supabase continuam necessárias.

IMPORTANTE
- Comece no sandbox.
- O token deve ficar somente na Vercel.
- O token OAuth do Melhor Envio expira; a integração completa de produção
  precisa de renovação de token ou de um token válido administrado por você.
- Confira pesos e dimensões reais antes de vender.
