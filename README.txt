OFERTA CERTA V14 PRO

ARQUIVOS:
- index.html: loja pública
- admin.html: painel de cadastro
- style.css: visual completo
- app.js: catálogo e botão direto para o Mercado Livre
- admin.js: cadastro, edição, exclusão, imagem e contagem de cliques

COMO USAR:
1. Extraia o ZIP.
2. Envie todos os arquivos para o mesmo diretório no seu site.
3. Abra admin.html.
4. Cadastre o produto:
   - cole o link de afiliado https://meli.la/...
   - informe título, preço e categoria
   - envie a imagem
   - clique em Publicar produto
5. Abra index.html para ver a loja.

IMPORTANTE:
- O botão "Comprar no Mercado Livre" usa exatamente o link de afiliado cadastrado.
- Portanto, o cliente vai diretamente ao anúncio indicado pelo seu link.
- Os dados ficam salvos no localStorage do navegador.
- Para publicar para todos os visitantes com banco de dados real, a próxima etapa é conectar Firebase, Supabase ou uma API própria.
