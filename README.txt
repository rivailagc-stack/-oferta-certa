OFERTA CERTA V15 PRO

ARQUIVOS:
- index.html: loja pública
- admin.html: painel de cadastro
- style.css: visual completo
- app.js: loja, busca, galeria e botão Comprar
- admin.js: cadastro, várias fotos, capa, edição e backup
- vercel.json: configuração da Vercel

COMO INSTALAR:
1. Faça backup dos arquivos atuais no GitHub.
2. Substitua index.html, admin.html, style.css, app.js e admin.js pelos arquivos desta pasta.
3. Adicione vercel.json caso ainda não exista.
4. Aguarde a Vercel concluir o deploy.
5. Abra /admin.html e faça um teste com 2 fotos.

IMPORTANTE:
Esta versão usa localStorage. Os produtos ficam salvos no navegador em que foram cadastrados.
Para que todos os visitantes e aparelhos enxerguem o mesmo catálogo, será necessário ligar o site a um banco online, como Supabase ou Firebase.
